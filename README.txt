README
Сервис ProductSales
Сервис ProductSales - это приложение на языке Python, которое помогает пользователям находить доступные товары, включая квартиры, автомобили и товары на Kaspi.kz. 
Приложение предоставляет простой и удобный графический интерфейс на основе Tkinter, который позволяет пользователям вводить критерии поиска и получать соответствующую 
информацию.

Возможности
Удобный интерфейс: Приложение предлагает простой и интуитивно понятный интерфейс, созданный с использованием Tkinter для облегчения использования.

Поиск функционала: Пользователи могут искать квартиры, автомобили или товары Kaspi.kz, вводя соответствующие ключевые слова. Приложение динамически корректирует поля 
ввода в зависимости от выбранной категории поиска.

Динамические поля ввода: В зависимости от выбранной категории поиска (квартира, автомобиль или другое), приложение динамически генерирует поля ввода, чтобы
пользователи могли указать дополнительные параметры поиска.

Анализ данных: Приложение использует технику веб-скрапинга для извлечения данных из Krisha.kz, Kolesa.kz и Kaspi.kz. Для параллельных веб-запросов повышается 
производительность с использованием асинхронной библиотеки aiohttp.

Как использовать
Запуск приложения: Запустите приложение, выполнив предоставленный скрипт Python (main.py). Появится окно Tkinter.
Ввод критериев поиска: Введите желаемый поисковый запрос (например, "квартира" для квартир, "машина" для автомобилей) в поле ввода поиска.
Отправка поиска: Нажмите кнопку "Поиск" ("Search"), чтобы отправить ваш запрос. Дополнительные поля ввода появятся в зависимости от выбранной категории.
Указание дополнительных данных: Заполните дополнительные данные в соответствии с вашими критериями поиска. Для квартир вы можете указать опции аренды или покупки, 
количество комнат, диапазон этажей и квадратных метров. Для автомобилей вы можете предоставить информацию, такую как марка, модель, год, объем, пробег, коробка 
передач, тип кузова.
Выполнение поиска: Нажмите кнопку "Найти" ("Find"), чтобы начать поиск. Приложение получит данные с соответствующих веб-сайтов и отобразит результаты в списке.
Просмотр результатов: Результаты поиска будут отображены в списке. Если данные слишком длинные, они будут разделены на несколько строк для удобства восприятия.
Повторение или выход: Вы можете выполнять несколько поисков, вводя новые критерии и снова нажимая кнопку "Поиск". Для выхода из приложения закройте окно Tkinter.

Зависимости
Приложение зависит от следующих библиотек Python:

aiohttp: Для асинхронных HTTP-запросов во время веб-скрапинга.
requests: Для выполнения HTTP-запросов.
beautifulsoup4: Для парсинга HTML-контента.
selenium: Для веб-скрапинга Kaspi.kz с использованием безголового браузера.
webdriver_manager: Для управления установкой драйверов браузера.
